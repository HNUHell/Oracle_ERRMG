# Database Error Messages

## 前言介绍
- 文档通过翻译文档、搜索资料及测试实践对Oralce的异常做汇总、分析导致异常的各类原因及对应的解决方法，并备注一些知识点以助学习了解。
- 文档内容持续更新，作者在时间许可情况下按``遇到的异常>感兴趣的异常>常见的异常>序号从小到大的异常``此顺序进行分析更新，暂时是作者的oracle异常学习记录。
